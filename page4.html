The Internet

the internet is a network of networks, its compromised of hardware, software, and protocols that allows computers and other devices to exchange information.

Origin of Internet

The internet was first conceived of during cold war. The american government was looking for ways to communicate in the vent of a nuclear attack.

Servers

Web servers are special computers that are constantly connected to the internet, most individuals/companies don't have their own data centers/servers but lease space from a web hosting service.

Clients

A client is a device that requests info from the server.

Web browsers: a cilent software that provided a way to look and interact with all the info in the web.

Protocols

Set of rules and agreed upon standards that allow for exchange of information on the world wide web.

HTTP: a protocol for distributing information.

FTP: a protocol used to copy a file from one host to another

IP Address

device connected to the web its assigned an ip address

Web Development Process

The process of creating a website can be divided into 4 main parts:

  • creative brief
  • user experience
  • front end development
  • back end development

    • Creative Brief

    The creative brief outlines the needs of the client and the goals of the website.

    User Experience

    The UX designer uses a site map, wireframes and user testing to identify the most intutive experience.

    Front End Development

    The front end developer translates the wireframes into HTML and CSS code and adds interactions with javascript.

    Back End Development

    The key role of a back end developer is to ensure that the data requested by the front end system is delivered.
